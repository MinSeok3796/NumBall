# 숫자야구 게임 만들기

#### 숫자야구  소개 
숫자야구 게임은 1~9의 숫자들을 이용하여 랜덤한 숫자를 맞추는 게임입니다.

#### 게임 규칙
- 1~9까지 서로다른 n자리 숫자를 입력한다.
- 입력받은 숫자에 대한 스트라이크, 볼, 아웃의 정보를 제공한다.

__스트라이크 :__  숫자와 위치가 모두 일치할 때

__볼 :__ 숫자는 맞지만 위치가 다를 때

__아웃 :__ 일치하는 숫자가 없을 때


#### 구현 기능
1. 랜덤숫자 생성
2. 입력받은 숫자와 비교 및 결과 출력
3. 난이도 조절
4. 정답까지 도전횟수 기록보기
